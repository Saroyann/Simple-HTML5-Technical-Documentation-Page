Introduction

HTML (Hyper-Text Markup Languange) or HTML5 is the most used Markup languange for web development. Usually, the common use of HTML is to arrange paragraph sections, headings, or links on web pages. But, even though the structure is like coding, please note that HTML is not a programming language.HTML is not considered a programming language because it cannot provide dynamic functions. Today, this markup language is considered an official web standard, maintained by the World Wide Web Consortium (W3C), which is also in charge of releasing regular HTML updates.
for More specific, HTML functions is:

  • Create web pages
  • Displays various information in an Internet browser.
  • Create a link to another web page with a certain code (hypertext).

HTML5 Anatomy

Before writing HTML5, you must first know about the anatomical structure of HTML5.

In general, HTML5 has the following anatomical structure:

<!DOCTYPE html>
<html>
<head>
<meta>
<title>Your Document Title</title>
</head>
<body>
</body>
</html>

<!DOCTYPE html>

<!DOCTYPE html> is a declaration or to identify the type of HTML document used so that the browser can determine how to treat the code.

<html> ... </html>

The <html> tag in HTML is the root tag used to mark the start and end of an HTML document. All elements used in web pages must be placed inside html tags.

<head>...</head>

<head> in HTML is used to store meta information about the document. The information contained in the head will not be displayed on web pages but is still important for browsers and search engines.

<meta>

<meta> tags are used to provide information about the HTML document itself. <meta> are used in the head section of a web page to provide additional information about that page to browsers, search engines, or other applications that access the page.

Some examples of common uses of meta tags:

  • <meta charset="UTF-8">is used to define the type of character used in the document.

  • <meta name="description" content="This is a web page description">is used to provide a summary of what a web page is about, which is often used by search engines as a description in search results.

  • <meta name="keywords" content="HTML, CSS, JavaScript">is used to provide keywords relevant to the web page content, which are used by search engines to determine the relevance of the web page to searches.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">is used to ensure that web pages display properly on mobile devices by adjusting the screen size and display scale.

  • <meta name="robots" content="index,follow">is used to tell search engines whether a web page should be indexed or followed.

<title>...</title>

The <title> tag in HTML is used to specify the title of a web page. This title will be displayed in a browser tab or window, and used by search engines as the title of search results.

<body>... </body>

The <body> tag in HTML is used to store content that will be displayed on a web page. All elements that will be displayed to users such as text, images, forms, links, etc. must be placed inside the <body> tag.

Tags

Tags in HTML are labels used in HTML documents to mark sections of content. Tags are usually written in curly brackets and begin with a "<" sign and end with a ">" sign. For example: <p>, <h1>, <div>. Each tag has a different function, such as <p> to create paragraphs, <img> to add images, <a> to create links, etc. Some tags also require an attribute that is used to provide further properties, such as <img src="image.jpg"> to specify the location of an image file.

Tags in HTML are divided into two types, namely:

  • The opening tag is used to start an element, for example: <p>
  • The closing tag is used to end an element, for example: </p>

Some tags also have a special property called self-closing tags. that don't require a closing tag, for example: <br> or <img>.

In HTML, some of the tags that are often used are:

  • <html>: This tag is used to mark the beginning of an HTML document.
  • <head>: This tag is used to mark the part of the HTML document that contains information about the document itself, such as the title, meta description, and links to stylesheets.
  • <title>: This tag is used to specify the title of the document to be displayed on the browser tab.
  • <body>: This tag is used to mark the part of the HTML document that contains content that will be displayed on a web page.
  • <p>: This tag is used to display paragraphs.
  • <div>: This tag is used to divide content into sections that can be assigned different properties and styles.
  • <img>: This tag is used to add an image.
  • <a>: This tag is used to create a link.
  • <ul>: This tag is used to create a list with bullet points.
  • <ol>: This tag is used to create a numbered list.
  • <li>: This tag is used to add items to the list.
  • <form>: This tag is used to create the input form.
  • <input>: This tag is used to create an input field on the form.
  • <label>: This tag is used to add a label to the input field.
  • <select>: This tag is used to make a selection from among the available options.

Some examples of HTML tags above are just some of the tags that are often used to build a website, in detail about the tags in HTML you can access via the following link: https://www.w3schools.com/tags/default.asp

Multimedia

In HTML, multimedia refers to elements that can be used to display various types of media, including video, audio, and images.

Some common HTML tags for embedding multimedia into a web page include:

  • <video>: used to embed a video file into a web page.
  • <audio>: used to embed an audio file into a web page.
  • <img>: used to embed an image file into a web page.

There are several ways to add multimedia, such as images, audio, and video, to an HTML document. Here are a few examples:

  • To add an image:
    • <img src="image.jpg" alt="A description of the image">
  • To add audio:
    • <audio controls>
    <source src="audio.mp3" type="audio/mpeg"> </audio>
  • To add video:
    • <video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">

You can also adjust the width and height of the multimedia and if you want to support multiple format you can add multiple source of file types in source tag.

Form

In HTML, a form is used to collect user input. The <form> element is used to create the form, and various form controls, such as text fields, checkboxes, and submit buttons, are placed within the form.

Here's an example of a simple form that includes a text field, a password field, and a submit button:

<form action="/submit-form" method="post">
<label for="username">Username:</label>
<input type="text" id="username" name="username">

<label for="password">Password:</label>
<input type="password" id="password" name="password">

<input type="submit" value="Submit">
</form>

The <form> element has two important attributes: action and method. The action attribute specifies where the form data should be sent when the form is submitted, and the method attribute specifies the HTTP method (either "get" or "post") to be used when sending the form data.

The <label> element is used to provide a text description for form controls. The for attribute is used to associate the label with a specific form control. The <input> element is used to create the form controls, and the type attribute specifies the type of control (e.g. "text", "password", "checkbox", "submit"). The id and name attribute of the input element will be used when you want to get the value with javascript or when you want to submit the form and server can identify each input

To find out more about forms in HTML, visit the following link: https://www.w3schools.com/html/html_forms.asp

Table

In HTML, the <table> element is used to create a table. A table is made up of rows and columns, and each row is represented by a <tr> (table row) element, while each column is represented by a <td> (table data) element.

Here's an example of a simple table:

<table>
<tr>
<th>Header 1 </th>
<th>Header 2 </th>
</tr>
<tr>
<td>Row 1, Cell 1 </td>
<td>Row 1, Cell 2 </td>
</tr>
<tr>
<td>Row 2, Cell 1 </td>
<td>Row 2, Cell 2 </td>
</tr>
</table>
